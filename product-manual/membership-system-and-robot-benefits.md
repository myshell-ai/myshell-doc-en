# Membership System and Robot Benefits

### Overview

Users with any type of MyShell Pass NFT can gain more robot benefits by upgrading their membership level.

### Membership Levels

| Level       | Requirements                            | Benefits                                                                 |
| ----------- | -------------------------------------- | ----------------------------------------------------------------------- |
| Pre-Members | Registered members in any form         | Access to product features                                               |
| Official Members | Own MyShell Pass NFT                  | Participate in the crypto-economic system                               |
| Genesis Members | Own MyShell Genesis Pass NFT, limited to 10,000 | Access to powerful Beta features, easier access to platform benefits and preferential measures in the crypto-economic system |
